Support

Maintainers

HandsOnArchitects.com oversees the maintenance of Knot.x. To inquire about upgrades, fixes, and long-term support, kindly contact the core developers directly.

Discussions

Please don't hesitate to initiate discussions, pose inquiries, or share your ideas through GitHub Discussions.

Issues

You can submit feature requests and report bugs by creating issues on the Github Issues.

Updates

Knot.x blog page is the place you can find such information like tutorials and articles. We recommend also to follow our @knotx_project twitter account if you are interested in current news / updates.

Contributing

Bug fixes and feature contributions in the form of pull requests are welcome. If you have a non-trivial change in mind that you'd like to discuss, feel free to contact us on GitHub Discussions.
